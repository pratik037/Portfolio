Projects Summary

This section will give a small summary/brief of the projects that are mentioned above.

Live Unanimous Currency Interconversion (LUCI)

LUCI is used to detect currency notes using a camera (using OpenCv) and then fetches(from CurrencyLayer API) and displays (using Python) real time value in USD, Euro etc. This kind of application is used to prevent people from getting robbed while travelling internationally. Users can quickly check the value of their currency and then pay accordingly.

StreetVolunteer

SV is a cross platform mobile application developed using Flutter and Dart. It is a de-centralised platform for people who want to do good for the community. Users can post and join campaigns which can be helpful during the time of crisis. It also has a feature of audio QR, which can be used to send and recieve text message online/offline anytime in a certain range, which can prove helpful during rescue missions.

Memories

Memories is also a cross platform mobile application developed using Flutter and Dart. Memories makes use of the Google Maps API and it renders Google Maps on the homescreen. Users can then tap on the google map or search for any place using the search bar to add their memories of that place. It can be any text and images. Memories are then shown as red markers on Google Maps, which can act as a post trip timeline for its users.
